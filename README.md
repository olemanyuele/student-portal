# 🎓 Student Portal Web App

A simple and beautiful web-based student portal that allows students to sign up, log in, view assignments, check grades, and upload their submissions. It also includes an admin panel for managing student data and viewing submissions.


## 🌟 Features

### For Students:
- 👤 Sign Up & Login
- 📚 View Assignments
- 📝 View Grades
- 📤 Upload Assignment Files

### For Admins:
- 🔐 Admin Login
- 📋 View and Edit Grades & Assignments
- 📥 View Student Submissions

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (with `localStorage`)
- [GitHub Pages](https://pages.github.com) for hosting (or Netlify)

## 🚀 How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser to get started.
3. Use the links to navigate between:
   - Student Sign Up
   - Student Login
   - Student Dashboard
   - Admin Login
   - Admin Panel
